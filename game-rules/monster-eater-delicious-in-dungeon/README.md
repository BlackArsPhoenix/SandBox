# Monster Eater: A Delicious in Dungeon Board Game

Настольная игра по манге/аниме **«Подземелье вкусностей»** (Dungeon Meshi).

| | |
|---|---|
| Японское название | モンスターイーター ～ダンジョン飯 ボードゲーム～ |
| Издатель (JP) | Arclight Games (2022) |
| Издатель (EN) | Yen Press (26 мая 2026) |
| Дизайн | Канай Сэйдзи (Seiji Kanai) |
| Идея | Судзуки Гинъитиро (Monster Maker) |
| Иллюстрации | Куи Рёко |
| Игроки / время / возраст | 2–5 / ~45 мин / 10+ |

Соревновательный данжен-кроулер: ведёте партию, исследуете верхний и нижний уровни, шлёте монстров соперникам, готовите побеждённых ради еды и победных очков. Игра заканчивается победой над Сиселем (Безумным магом).

## Кратко о правилах

1. Ход: **Бой** или **Снабжение** → (после боя) **Исследование** → **Встреча** → **Отдых**.
2. Исследование картами «Пройдено X%»; при ≥100% на уровне — босс (Фалин / Сисель).
3. Монстр на вашем подземелье блокирует исследование, пока его не победите.
4. После победы — готовка (2D + кулинарная сила ≥ сложности) → еда и очки.
5. Конец: победа над Сиселем → подсчёт очков (предметы, блюда, боссы, деньги/3, штрафы за недобор %).
