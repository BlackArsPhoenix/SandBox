# Винрейты бойцов BattleCON (русская локализация)

## Важно

Публичной официальной таблицы винрейтов по человеческим рейтинговым партиям **нет**.
BattleCON Online показывает только *личный* винрейт после 15 побед за бойца.

Ниже — **единственный найденный скачиваемый количественный датасет**: результаты AI vs AI
из репозитория [yaronra/battlecon-ai](https://github.com/yaronra/battlecon-ai)
(`logs/free_for_all`, ~1 партия на матчап). Это **не** винрейты живых игроков.

Русская локализация GaGa: **Погибель** (32) + **Странники** (10) + **Испытания** (10) = **52** бойца.
AI-данные есть у **22** из 52; у остальных персонажей симуляции не было.

## Источники

- Состав RU: GaGa Games (Погибель / Странники / Испытания); roster Devastation Remastered и Trials Remastered (Level 99); Wanderers of Indines.
- AI winrates: https://github.com/yaronra/battlecon-ai `logs/free_for_all`
- Локальные заметки сообщества: Tesera (dizastra) — Руника ~87%, Мармелли ~30% (не глобальная статистика).
