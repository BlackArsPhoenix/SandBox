# Massive Darkness 2: Massive Darkness Upgrade Pack

Набор обновления: позволяет использовать миниатюры и контент Massive Darkness 1
(и её дополнений) в правилах Massive Darkness 2: Hellscape («Кромешная тьма: Преисподняя»).

Отдельного «полного» рулбука у пака нет — нужны правила MD2 + буклет кампании/пролога из коробки пака.

## Что даёт набор

- Обновлённые карты героев MD1 под классы MD2 (часть планшетов двусторонние)
- Карты врагов MD1 как Mobs / Roaming Monsters / Bosses
- Boss Dashboards (в т.ч. Abyssal Demon, Hellephant, Spearmaiden+Cliffbreaker Cyclops)
- Prologue Campaign Booklet (конвертация квестов MD1 и Crystal & Lava) + campaign cards
- KS-издание: больше карт (до ~369), retail — меньше

## Как играть

1. Правила MD2: `../_shared/massive-darkness-2/Kromeshnaya-tma-Preispodnyaya-rules-ru.pdf`
2. FAQ MD2: `../_shared/massive-darkness-2/MD2-FAQ-v1.1-en.pdf`
3. Карты/буклет из Upgrade Pack подменяют/дополняют контент MD1 для совместимости с MD2.
